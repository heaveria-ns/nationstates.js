Class Dispatch

A class to handle creating, editing, and deleting dispatches in more high-level functions.

example

const methods = new Dispatch(api, 'nation', 'password', Mode.add);

param API

The API object to enforce rate limiting and user agents.

param nation

Your nation name without prefixes. Used for authenticating with the NationStates API.

param password

Your password. Used for authenticating with the NationStates API.

param action

Enumerator to specify the action to be taken.
