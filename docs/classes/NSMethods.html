Class NSMethods

As opposed to building requests manually, this class has built-in methods for easily accessing and handling common information one looks for. You do not build, send, or parse requests manually.

example

const methods = new NSMethods(api);

param client

The client object to enforce rate limiting and user agents.

downloadDumpAsync

  • downloadDumpAsync(type: string, directoryToSave: string, options?: IDumpOptions): Promise<NSMethods>
    • Defined in src/ns_methods/ns_methods.ts:93

    Download the nation data dump from the client. Note that it can take a while to download the dump, especially for nations or when converting to JSON. Future feature: Decode utf-8 within the dump.

    example

    methods.downloadDump('Testlandia', {extract: true, deleteXMLGz: true, deleteXML: true, convertToJson: true}); // Returns just a .json file

    Parameters

    • type: string

      Either 'nation' or 'region'

    • directoryToSave: string

      The directory to save the dump to. Should be ended by a slash. Ex: "./downloads/"

    • Optional options: IDumpOptions

      If left blank, just downloads the {type}.xml.gz file.

    Returns Promise<NSMethods>
