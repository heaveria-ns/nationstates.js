Class RequestBuilder

Build a request to your specifications! Usage:

  • (1) Define the architecture of a https request before it sent to the API.
  • (2) Access and modify the response of a request.
example

const request = await new RequestBuilder(api).addNation('testlandia').sendRequestAsync(); console.log(request.body);

param api

The API instance to use. Used to enforce the rate limit and user agent.
